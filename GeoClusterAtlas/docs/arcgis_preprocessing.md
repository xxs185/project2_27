# ArcGIS 前处理说明

这个项目不直接绑定 ArcGIS 软件操作，但默认你会在 ArcGIS 或 ArcGIS Pro 里完成前处理，再把结果导出为表格接入本仓库。

## 推荐的前处理顺序

### 1. 统一空间参考

- 研究区边界、栅格、矢量图层保持同一坐标系
- 分辨率、分析范围尽量提前统一

### 2. 固定分析单元

- 可以按网格、街道、地块或行政单元进行汇总
- 不建议中途切换分析单元口径

### 3. 抽取解释变量

- 点状数据可转成密度、数量或邻近统计
- 线状数据可转成距离、通达性或覆盖比例
- 面状数据可转成占比、类型统计或强度值
- 栅格数据可按单元汇总成均值、中位数或极值

### 4. 导出建模表

建议导出的表至少包含：

- `单元编号`：空间单元唯一标识
- `特征1` 到 `特征10`：解释变量
- `目标值`：建模目标

### 5. 回图阶段

建模完成后，可将下面这些结果回接 ArcGIS：

- 预测值
- 残差
- 随机森林重要性结果
- SHAP 贡献结果

## 字段口径建议

- 字段名简洁，不混用空格和特殊符号
- 目标字段不要和解释变量存在直接计算关系
- 编号类字段不要直接送入模型
- 如果后续需要空间连接，务必保留稳定的主键字段
