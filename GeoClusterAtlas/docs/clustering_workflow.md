# 聚类模块说明

这个模块放在 ArcGIS 前处理和随机森林建模之间，主要用来做一件事：  
先把空间单元分成若干相对接近的类型，再看不同类型之间的差异和后续解释结果。

## 模块作用

- 为后续制图提供分区标签
- 为结果解读提供分组框架
- 帮助发现不同单元之间是否存在稳定结构

## 当前实现

项目里默认提供的是 KMeans 聚类流程，输入是一张已经整理好的建模表。

输入要求：

- 一行对应一个空间单元
- 一列对应一个变量
- 特征字段默认识别 `特征1` 到 `特征10`

输出内容：

- `cluster_metrics.csv`：不同 `k` 下的评估结果
- `cluster_assignments.csv`：每个样本在不同 `k` 下的类别标签
- `best_k_summary.json`：自动选出的参考聚类数
- `cluster_profile_best_k.csv`：最佳聚类数下的各类特征均值
- `cluster_centers_best_k.csv`：最佳聚类数下的聚类中心
- `elbow_curve.png`、`clustering_scores.png`：评估图表

## 推荐用法

### 1. 先跑聚类

先判断数据内部有没有比较稳定的分组结构。

### 2. 再做建模

随机森林和 SHAP 负责解释变量作用，聚类负责说明这些作用可能落在哪些类型上。

### 3. 最后回图

把聚类标签、典型类别特征和模型结果一起接回 ArcGIS，可以更方便做分区解读。
